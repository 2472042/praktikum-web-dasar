Homepage / Program Studi / Program Sarjana Teknik Informatika

Mahasiswa Program Sarjana Teknik Informatika akan mempelajari ilmu computer science serta implementasi praktis software engineering. Keahlian utama dengan keahlian-keahlian khusus yang dapat diambil sesuai bidang peminatan, yaitu:

  1. Game & Mobile Apps
  2. Webs & Multimedia
  3. Intelligent Computing
  4. Network Security

Aktivitas Akademik

Fokus utama perkuliahan adalah dalam ekahlian pengembangan software (software methods and technologies). Proses pembelajaran seimbang antara konseptual dan praktik di laboratorium, menyiapkan mahasiswa agar siap berkarya secara maksimal di dunia kerja/usaha setelah menyelesaikan kuliah. Lulusan akan memiliki keahlian pengembangan aplikasi mobile/desktop application, berbasis intelligent computing & network security.

Masa Depan dan Karir

Lulusan Program Sarjana Teknik Informatika sudah banyak yang berhasil menjadi wirausahawan, maupun bekerja di berbagai perusahaan di tingkat nasional, multinasional, bahkan skala internasional. Masa tunggu lulusan untuk memperoleh pekerjaan atau menjadi wirausahawan juga terhitung relatif singkat. Lulusan program ini dapat berkarier dalam bidang/ sebagai:

  • Teknopreneur: IT solutions, independent software vendor
  • IT Architech: system analyst
  • Software engineer: mobile developer, desktop developer, NET developer, JAVA developer, dan web developer, intelligent system developer
  • Database engineer: database developer/designer, database administrastor
  • System engineer: network engineer, security engineer
